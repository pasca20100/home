3.1 Introduction

This document describes the steps involved in creating a continuous gender measurement index, based on the cultural practices of the French in 2018 (EnquÃªte sur les Pratiques Culturelles, 2018).

\[I_{1j} = \sum_{k=1}^{Z} w_{1k} \cdot X_{k j}\]

In this expression, \(I_{1j}\) denotes the index for individual \(j\), while \(w_{1k}\) represents the weight associated with each cultural variable \(X_{kj}\). The summation encompasses all \(Z\) cultural variables, allowing us to capture the overall cultural engagement of the individual.

3.9 Building The Index

Show the code 
# Initialiser un vecteur pour stocker l'indice de chaque individu
ra_data$indice_culturel <- 0

# Parcourir chaque individu
for (i in 1:nrow(ra_data)) {
  
  # Initialiser l'indice de l'individu Ã  0
  indice_individu <- 0
  
  # Parcourir chaque colonne de pratiques (les colonnes de 3 Ã  11)
  for (pratique in 3:27) {
    
    # RÃ©cupÃ©rer la rÃ©ponse de l'individu pour cette pratique (0 ou 1)
    reponse <- ra_data[i, pratique]
    
    # Si la rÃ©ponse est 1, ajouter la coordonnÃ©e de la modalitÃ© correspondante Ã  l'indice
    if (reponse == 1) {
      
      # CrÃ©er le nom de la modalitÃ© (par exemple "knitting_1" ou "knitting_0")
      nom_modalite_1 <- paste0(names(ra_data)[pratique], "_1")
      nom_modalite_0 <- paste0(names(ra_data)[pratique], "_0")
      
      # Trouver la coordonnÃ©e associÃ©e Ã  la modalitÃ© correspondante
      if (nom_modalite_1 %in% modalites_coord$Modalite) {
        indice_individu <- indice_individu + modalites_coord$Coord_Dim2[modalites_coord$Modalite == nom_modalite_1]
      }
      if (nom_modalite_0 %in% modalites_coord$Modalite) {
        indice_individu <- indice_individu + modalites_coord$Coord_Dim2[modalites_coord$Modalite == nom_modalite_0]
      }
    }
  }
  
  # Assigner l'indice calculÃ© Ã  l'individu
  ra_data$indice_culturel[i] <- indice_individu
}
####Normalisation

# Calculer les valeurs minimum et maximum de l'indice
min_indice <- min(ra_data$indice_culturel, na.rm = TRUE)
max_indice <- max(ra_data$indice_culturel, na.rm = TRUE)

# Normaliser l'indice
ra_data$indice_culturel_normalise <- (ra_data$indice_culturel - min_indice) / (max_indice - min_indice)
